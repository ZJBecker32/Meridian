# Meridian Map Studio

Generate clean, customizable map visuals from OpenStreetMap data.

## Overview
Meridian is a lightweight, client-side web app that creates stylized map graphics from real-world locations. Simply enter a place, adjust the radius, and generate a clean, printable map instantly.

Built using OpenStreetMap data and the Overpass API.

## Features
- Search any city or location
- Adjustable map radius
- Clean, minimal map styling
- Fast, browser-based rendering (no backend required)
- Export-ready visuals for design and print use

## How to Use
1. Open the app
2. Enter a location
3. Adjust your settings (radius, etc.)
4. Generate your map

## Notes
- Map data is sourced from OpenStreetMap and may be subject to API rate limits
- Performance may vary depending on query size and location
